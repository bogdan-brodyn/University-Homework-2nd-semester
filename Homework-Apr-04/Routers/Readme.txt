You may find .exe file by path:
- ConsoleApplication\bin\Debug\net8.0\ConsoleApplication.exe

Program must be run with two arguments:
- the first must declare path to file to read graph from;
- the second must declare path to file to write result min spanning tree to;

Paths may be absolute or relative.

For example (being in directory Routers) you may run:
ConsoleApplication\bin\Debug\net8.0\ConsoleApplication.exe InputFileExample.txt OutputFileExample.txt
